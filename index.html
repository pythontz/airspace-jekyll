About Us

Pythontz is a positive peer learning community founded by Mr Anthony Faustine with aim to create a vibrant and diverse python and data scientists community in Tanzania.

Our mission is to foster the application of python programming across industries, learning centers, schools and community and building human capacity in data science, machine learning and artificial intelligence in Tanzania.

Our main agenda is to organize workshop, training and summer school that aim to expose interested people especially students to data science and machine learning using python. Kindly check our event page for more update.

WHY PYTHON

Python is a powerful, flexible, open source language that is easy to learn and easy to use.It has a unique combination of being both a capable general-purpose programming language as well as being easy to use for analytical and quantitative computing.

Python is great for backend web development, data analysis, artificial intelligence, and scientific computing. You can use Python to build productivity tools, games, and desktop apps
